## 🚀 Project Highlights

### 📊 [Adventure Works Sales Report](https://github.com/mrsaurabhtanwer/Adventure-Works-Sales-Report)
Interactive Power BI dashboard for AdventureWorks sales, showing trends, KPIs, and product performance insights.

### 🏬 [Superstore Dashboard](https://github.com/mrsaurabhtanwer/Superstore-dashboard)
Power BI dashboard to analyze Superstore’s performance across categories, regions, and time using dynamic filters.

### 🍕 [EDA - Pizza Restaurant Sales](https://github.com/mrsaurabhtanwer/EDA-Pizza-Restaurant-Sales)
Python-based exploratory analysis of a pizza restaurant’s sales data to uncover revenue trends and customer patterns.

### ✉️ [GEN-AI Cold Email Generator](https://github.com/mrsaurabhtanwer/GEN-AI-Cold-Email-Generator)
AI tool built with Streamlit and OpenAI GPT API to auto-generate professional cold emails based on user prompts.

### 🧠 [Customer Segmentation](https://github.com/mrsaurabhtanwer/Customer_Segmentation)
K-means clustering applied on customer data to identify distinct customer groups and improve targeted marketing.

### 🎵 [SQL Music Store Analysis](https://github.com/mrsaurabhtanwer/SQL-Music-Store-Analysis)
Advanced SQL queries used to analyze sales trends, top artists, and customer behavior from a music store database.

### 🦠 [COVID Data Analysis Project](https://github.com/mrsaurabhtanwer/Covid-Data-Analysis-Project)
Analysis and visualization of COVID-19 global data using Python and Tableau for insights into infection and recovery rates.
